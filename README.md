# School Hub ✧

A pastel-themed **student productivity dashboard** that helps students manage assignments, track deadlines, and stay organized.

School Hub features a **Notion-style layout with a sidebar and interactive to-do list**, built using **HTML, CSS, and Vanilla JavaScript**.  
The project demonstrates frontend development concepts such as **DOM manipulation, event handling, localStorage persistence, and responsive UI design**.

## ✨ Key Features

### Assignment Manager
Users can efficiently manage assignments by:

- Adding new tasks
- Setting due dates
- Marking tasks as completed
- Deleting tasks
- Highlighting important tasks with an **Urgent tag**

Completed tasks automatically appear with **line-through styling and faded appearance**.

---

### ⏳ Custom Countdown Timer

The dashboard includes a customizable countdown feature for tracking important events.

Users can:

- Enter a label for the event (example: Winter Break, Exams, Project Deadline)
- Select a target date
- Save the countdown

The remaining time is dynamically calculated using **JavaScript**.

---

### 🔎 Task Filtering

Tasks can be organized using filter options:

- **All** – shows all tasks
- **To Submit** – shows pending assignments
- **Completed** – shows finished tasks

This allows users to quickly focus on unfinished work.

---

### 📊 Assignment Statistics

The dashboard automatically tracks:

- Total tasks
- Tasks to submit
- Completed tasks
- Urgent tasks

These statistics update dynamically as tasks change.
